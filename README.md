# chat-application

Chat application that allows you to exchange messages in real time.

## Main Functions

- Real time messaging
- Authorization by name
- Browsing users in room

## Technologies

Front-End:

- ReactJS
- MaterialUI

Back-End

- NodeJS
- TypeScript
- SocketIO

Formatter:
- Prettier

Linter:
- TSlint

Code style:
- Google TypeScript Style

Testing:
- Jest

> Data may change during the development phase...
